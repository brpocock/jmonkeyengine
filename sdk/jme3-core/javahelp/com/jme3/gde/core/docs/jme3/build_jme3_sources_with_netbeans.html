Setting up a svn checkout of the jME3 Engine

You are welcome to try out the new jME3, and contribute patches and features! This document shows how to download, set up, build, and run the latest development version from the sources.

Note: In the following, always replace ”~” with the path to your home directory.

Downloading the Sources

Check out the sources from the repository.

  1. In jMonkeyPlatform go to Team > Subversion > Checkout
    1. Repository URL:
    2. You can leave user/pw blank for anonymous access.
  2. Click Next
    1. Repository Folders: “trunk/engine”
    2. Local Folder: ~/jMonkeyProjects
  3. Click Finish and wait.

The jme3 project opens in the Project window.

Build and Run

That's it!

  1. Right-click the jme3 project node and “Clean and Build” the project.
  2. In the Projects window, browse to the src/jme3_test folder.
  3. Right-click the file jme3_test/model/TestBox.java and choose “Run” to run a sample.
    1. In the sample application, use the mouse and the AWSD keys to move around the test object.
    2. Press escape to quit the sample application.
