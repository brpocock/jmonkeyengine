Geometry

com.jme3.scene.Geometry

  • A Geometry is a visible Spatial.
  • A Geometry contains a Mesh.
  • You can attach a (visible) Geometry to an (invisible) Node.
  • A Geometry can be saved and loaded using the AssetManager.

The Mesh in the Geometry can be a Shape, Custom Meshes, or loaded 3D models.
