Showing the Wireframe

WIP

I assume that you

  1. Have loaded your model.
  2. Have created one Material object for the mode, e.g. named m_model.
  3. Set the model's material to m_model.

Then you can add a switch to show the wireframe like this:

  1. Create a wire material, e.g. m_wire 
      Material m_wire = new Material(assetManager, "Common/MatDefs/Misc/WireColor.j3md");
  m_wire.setColor("Color", ColorRGBA.Green);
  2. Create a key input trigger that switches between the two materials: E.g. to toggle when the T key is pressed: 
        inputManager.addMapping("wireframe", new KeyTrigger(KeyInput.KEY_T));
    inputManager.addListener(actionListener, "wireframe");
  3. Now add the action listener and the toggle action 
      private () {
    @Override
    public void onAction( name, boolean pressed, float tpf) {
      if (name.equals("wireframe") && !pressed) {
        wireframe = !wireframe;
        if (!wireframe) {
          terrain.setMaterial(matWire);
        } else {
          terrain.setMaterial(matRock);
        }
      }
    }
  };
